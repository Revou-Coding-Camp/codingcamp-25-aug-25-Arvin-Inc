## 🎭 Animasi & Transisi

### CSS Animations
```css
/* Smooth hover effects */
.element {
  transition: all 0.3s ease;
  transform: translateY(0);
}

.element:hover {
  transform: translateY(-5px);
  box-shadow: 0 10px 25px rgba(0,0,0,0.2);
}

/* Keyframe animations */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
